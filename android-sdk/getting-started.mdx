# Getting Started with Android SDK

The Tolgee Android SDK provides powerful localization capabilities for your Android applications. It offers real-time translation updates, in-context editing, and seamless integration with the Tolgee platform.

## What is Tolgee Android SDK?

The Tolgee Android SDK is a comprehensive localization solution that enables you to:

- **Load translations dynamically** from Tolgee platform
- **Enable in-context editing** for translators and content managers
- **Support multiple languages** with automatic switching
- **Handle plural forms** and ICU message formatting
- **Work offline** with cached translations
- **Integrate seamlessly** with Android development workflows

## Quick Start

1. **Install the SDK** - Add the dependency to your `build.gradle`
2. **Configure Tolgee** - Set up your project credentials
3. **Initialize SDK** - Initialize Tolgee in your Application class
4. **Start Translating** - Use Tolgee in your activities and fragments
