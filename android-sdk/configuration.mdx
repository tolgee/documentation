# Configuration

## Basic Configuration

```kotlin
Tolgee.init(
    context = applicationContext,
    apiKey = "your-api-key",
    apiUrl = "https://app.tolgee.io",
    projectId = "your-project-id"
)
```

## Configuration Options

### Required Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `apiKey` | String | Your Tolgee API key |
| `projectId` | String | Your Tolgee project ID |

### Optional Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `apiUrl` | String | `https://app.tolgee.io` | Tolgee server URL |
| `language` | String | System language | Default language |
| `fallbackLanguage` | String | `en` | Fallback language |
| `devMode` | Boolean | `false` | Enable development mode |
| `cacheEnabled` | Boolean | `true` | Enable caching |
| `cacheExpiration` | Long | `24 * 60 * 60 * 1000` | Cache expiration (ms) |

## Advanced Configuration

### Custom Configuration

```kotlin
Tolgee.configure {
    apiKey = "your-api-key"
    apiUrl = "https://app.tolgee.io"
    projectId = "your-project-id"
    
    // Language settings
    defaultLanguage = "en"
    fallbackLanguage = "en"
    availableLanguages = listOf("en", "es", "de", "fr")
    
    // Caching
    cacheEnabled = true
    cacheExpiration = 24 * 60 * 60 * 1000 // 24 hours
    maxCacheSize = 50 * 1024 * 1024 // 50MB
    
    // Development
    devMode = BuildConfig.DEBUG
    
    // Network
    timeout = 30_000 // 30 seconds
    retryCount = 3
}
```

### Environment-Specific Configuration

```kotlin
// Development
if (BuildConfig.DEBUG) {
    Tolgee.configure {
        devMode = true
        cacheEnabled = false
        apiUrl = "https://localhost:8080" // Local server
    }
} else {
    // Production
    Tolgee.configure {
        devMode = false
        cacheEnabled = true
        apiUrl = "https://app.tolgee.io"
    }
}
```

## Security Configuration

### API Key Management

```kotlin
// Store API key securely
object TolgeeConfig {
    const val API_KEY = BuildConfig.TOLGEE_API_KEY
    const val PROJECT_ID = BuildConfig.TOLGEE_PROJECT_ID
}

// Initialize with secure values
Tolgee.init(
    context = applicationContext,
    apiKey = TolgeeConfig.API_KEY,
    projectId = TolgeeConfig.PROJECT_ID
)
```
