## Tolgee web

Tolgee web package contains all general web related tools. Mainly connected to in-context translating as it is tightly coupled to the DOM and web environment. Tolgee exports everything from `@tolgee/core`, so you can just install this package if you use Tolgee on web.

It contains these basic parts:

- [`InvisibleObserver`](./plugins#invisibleobserver)
- [`ContextUi`](./plugins#contextui)
- [`DevBackend`](./plugins#devbackend)
- [`BackendFetch`](./plugins#backendfetch)
- [`LanguageStorage`](./plugins#languagestorage)
- [`LanguageDetector`](./plugins#languagedetector)
- [`TextObserver`](./plugins#textobserver)
- [`BrowserExtensionPlugin`](./plugins#browserextensionplugin)
- [Other tools](./other-tools)
- \+ everything from [`@tolgee/core`](../core_package/about)

Plus bundled plugins for specific use cases:

- [`InContextTools`](./bundled-plugins#incontexttools)
- [`DevTools`](./bundled-plugins#devtools)
