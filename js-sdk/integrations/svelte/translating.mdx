There are two ways how to translate with Tolgee.

## `getTranslate` method

Other option how to translate is using of [`getTranslate`](./api#function-gettranslate) function. You can use it this way:

```svelte
<script>
  import { getTranslate } from "@tolgee/svelte";

  // t is store
  const { t } = getTranslate();
</script>

<!-- use the translation function -->
{$t('key', 'Default value')}
```

<TFunctionHint />

## The `T` component

An alternative way is to use [`T` component](./api#t-component), which has a very similar use case and API to the `t` function mentioned above. 

```svelte
<script>
  import { T } from "@tolgee/svelte";
</script>

<T 
  keyName="translation_key" 
  defaultValue="User has {count} points"
  params={{ count: 10 }}
/>
```

### Using namespaces

The `getTranslate` function can be also used for loading additional namespaces.

```ts
const { t, isLoading } = getTranslate('common');
```
