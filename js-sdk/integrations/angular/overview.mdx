Tolgee provides an Angular SDK built on top of [`@tolgee/web`](../../api/web_package/about). With [`@tolgee/ngx`](https://www.npmjs.com/package/@tolgee/ngx), you can connect an Angular 20 application to Tolgee using Angular's current standalone and application-config APIs.

The SDK gives you:

- `provideTolgee` for registering Tolgee during app bootstrap
- `TranslatePipe` and `TDirective` for template translations
- `TranslateService` for imperative translations and language switching
- `namespaceResolver` for loading namespaces before lazy routes render

- Follow the [installation guide](./installation) to add the Tolgee Angular SDK to your existing Angular application.
- You can also check the [sample Angular app](https://github.com/tolgee/tolgee-js/tree/main/testapps/ngx/projects/sampleapp) for a complete integration example.
