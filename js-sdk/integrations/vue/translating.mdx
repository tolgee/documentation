There are several ways how to translate with Tolgee.

## Global `$t` function

If you use [`VueTolgee`](./api#vuetolgee-plugin) plugin, you can use `$t` function in any component.

```html
<template>
  <div>{{ $t('this_is_a_key') }}</div>
</template>
```

## `useTranslate` composable

The other option how to translate is [`useTranslate`](./api#composable-usetranslate) composable.

```html
<template>
  <div :title="t('key')">...</div>
</template>

<script setup>
  import { useTranslate } from '@tolgee/vue';

  const { t } = useTranslate();
</script>
```

## The `T` component

Another way is to use [`T` component](./api#t-component), which has very similar API to the [`t` function](#t-function). 

```html
<template>
  <T
    keyName="translation_key" 
    defaultValue="User has {count} points"
    :params="{ count: 10 }"
 />
</template>

<script setup>
  import { T } from '@tolgee/vue';
</script>
```

## Using namespaces

The [`useTranslate`](./api#composable-usetranslate) composable can be also used for loading additional namespaces.

```ts
const { t, isLoading } = useTranslate('common');
```
