For language management, use [`useTolgee`](./api#hook-usetolgee) hook. With passed [`language`](/docs/sdk/api/core_package/events) event, so the component will get re-rendered when this event occurs.

```jsx
import { useTolgee } from '@tolgee/react';

export const LangSelector = () => {
  const tolgee = useTolgee(['language']);

  return (
    <select
```
