First, you need to install the plugin from the [Figma Community](https://www.figma.com/community/plugin/0) by clicking on the **Try it out** button.
Alternatively, you can find the plugin in Figma by opening the Resources > Plugins, searching for "**Tolgee**", and then click **Run**.
