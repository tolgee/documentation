## Machine translation settings
Machine translation (MT) uses external providers to translate base text to currently edited language.
Number of machine translations that you can use is limited by credits which are shown in language settings.

![Machine translation settings](/img/docs/platform/machine_translation.png)

To enable machine translation when self-hosting check out [configuration](/platform/self_hosting/configuration#machine-translation).
