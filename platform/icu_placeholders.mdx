Tolgee platform is able to convert placeholders from various formats to the ICU message format and vice versa.
This concept helps you to be able to use export data to different formats and get desired formatting when using the
same translations in different applications.

The Tolgee Universal ICU Placeholders are also recognized by the translation editor, and so they're highlighted to
provide convenient translation experience also for right-to-left languages.
